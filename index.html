About   Me

Hello There, and thank you for visiting! My name is Jaime Sandoval, and I'm a Software Developer based in Orange County, California.

I truly enjoy using the combination of creativity and logic to create applications. Whether the coding involves implementing an API or debugging, I'm all for it. If it helps the team get closer to our goals in any way, I'll do my best to get it done.

While attending LearningFuze coding boot camp, I was immersed in technologies used for building Full Stack Applications (900+ hours coding). LearningFuze gave me a solid foundation from which to work, and I've been busy using that foundation ever since.

Staying up to date with current dev trends is another passion of mine. I do that in a variety of ways, such as attending tech meetups, listening to dev podcasts, and finding interesting Software Developers to follow on Twitter.

I'd love to bring my skills to a company, and I look forward to being part of a Dev Team. If you'd like to contact me, please feel free to do so. Let's help each other reach our goals with clean code and teamwork.
